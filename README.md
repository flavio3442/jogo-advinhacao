# 🎯 Jogo de Adivinhação em Python

Um jogo simples em Python onde o jogador deve adivinhar um número aleatório entre 1 e 100.  
Ideal para iniciantes praticarem lógica de programação, entrada de dados, loops e tratamento de erros.
